# 📊 Portfolio Data Analisis - Nuwun Tea

Selamat datang di repositori portfolio analisis data untuk **Nuwun Tea**! Proyek ini menampilkan analisis data mendalam menggunakan **Power BI** dan berbagai teknik visualisasi data untuk memberikan insights berharga tentang bisnis Nuwun Tea.

**Disclaimer** : Data yang digunakan merupakan data artifisial yang berasal dari Claude Ai. 

## 🎯 Tentang Proyek

Proyek ini adalah sebuah studi kasus analisis data komprehensif yang fokus pada:
- **Analisis Penjualan**: Tren penjualan, performa produk, dan segmentasi pelanggan
- **Visualisasi Data**: Dashboard interaktif menggunakan Power BI
- **Business Intelligence**: Insights strategis untuk pengambilan keputusan bisnis

## File Power BI (.pbix)

Folder **Power BI** berisi workbook Power BI yang mencakup:

### Dashboard Utama
- **Dashboard Penjualan**: Ringkasan KPI utama, tren penjualan bulanan, dan performa kategori produk
- **Analisis Produk**: Produk terlaris, margin keuntungan, dan analisis inventaris
- **Segmentasi Pelanggan**: Distribusi pelanggan, lifetime value, dan pola pembelian
- **Analisis Regional**: Performa per wilayah, pertumbuhan year-over-year


##  Teknologi yang Digunakan

- **Power BI**: Untuk visualisasi data dan pembuatan dashboard interaktif
- **Microsoft Excel**: Untuk persiapan dan cleaning data
- **DAX (Data Analysis Expressions)**: Untuk custom calculations dan measures

## Metrik Utama

Dashboard mencakup analisis:
- **Revenue Growth**: Pertumbuhan pendapatan year-over-year
- **Sales Performance**: Pencapaian target penjualan
- **Customer Metrics**: Total pelanggan aktif dan churn rate
- **Product Analysis**: SKU performance dan product mix

## Insights Kunci

Beberapa insights yang dapat diperoleh dari dashboard:

Identifikasi tren penjualan dan musiman
Produk mana yang paling profitable
Segmentasi pelanggan berdasarkan perilaku pembelian
Peluang pertumbuhan di wilayah tertentu
Rekomendasi strategis untuk optimasi bisnis

Proyek ini dipublikasikan untuk tujuan portfolio dan pembelajaran.
